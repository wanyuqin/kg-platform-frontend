# ADR-0015：飞书来源：单文档注册、平台侧正文只读、事件+轮询双通道（P2）

- **状态**：已采纳
- **日期**：2026-07-04
- **来源**：设计文档 第三章（三项新增决策）

## 背景

飞书同步通道（P2）需要决定注册粒度、编辑权归属与同步机制。调研确认：飞书按文件订阅可行，但事件不含内容且无实时性 SLA。

## 决策

1. **单文档注册**：只做按文档注册，不做目录挂载（目录内容不可控，批量注册质量无法保证）；
2. **平台侧正文只读**：飞书来源的知识在平台侧不可编辑正文，运营只改飞书原文，经重拆对齐（ADR-0009）同步——避免双头编辑冲突；
3. **事件 + 轮询双通道**：事件订阅做低延迟触发，轮询兜底必需（事件无内容、无 SLA）。

另：Markdown 上传通道先选类型 + 前置校验，不过则拒收并给模板（设计 3.1）。

## 理由

单一编辑源是同步正确性的前提；轮询兜底是调研结论（事件不可靠）而非偏好。
