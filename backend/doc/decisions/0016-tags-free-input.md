# ADR-0016：tags 自由输入、可为空（取代受控词表）

- **状态**：已采纳
- **日期**：2026-07-04（MVP 技术设计阶段变更）
- **来源**：技术设计文档 1.2 变更①（原口径：设计文档 4.2 受控词表）

## 背景

设计文档原定 tags 走受控词表、不允许自由生成、不允许空值入库；MVP 阶段无人维护词表，词表反而成为录入阻力。

## 决策

tags 由用户自由输入，允许为空。DB 用 `TEXT[]` + GIN 索引支持过滤。

## 影响

- 元数据 tags 字段口径变更（"不允许空值入库"不再适用）；
- 表单录入页去掉词表选择器；
- P2"模板与词表"模块的词表维护取消；
- tag 不能进存储 URI（见 ADR-0019）。

## 理由

MVP 优先降低录入摩擦；检索主链路是语义检索，tag 只是辅助过滤，词表纯度收益有限。
