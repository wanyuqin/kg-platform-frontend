# ADR-0009：原文更新采用全量重拆 + 智能对齐

- **状态**：已采纳
- **日期**：2026-07-03
- **来源**：设计文档 4.1.4（重拆对齐画板）；P2 实现

## 背景

飞书原文更新后，需要把变化同步到已拆出的多条知识。备选：段落级 diff 定位变化，或全文重拆后与存量对齐。

## 决策

不做段落级 diff；原文更新触发**全量重拆**，重拆结果与存量条目**智能对齐**（内容相同保留 kid、变化的走更新、消失的走下架流程）。

## 理由

口语长文的段落边界不稳定，diff 对齐脆弱；全量重拆逻辑与首次导入同一条路径，实现简单且结果确定。kid 稳定性由对齐保证而非 diff。
