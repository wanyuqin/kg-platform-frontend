# ADR-0007：模板发布后冻结；六类类型与模板定稿

- **状态**：已采纳
- **日期**：2026-07-04（定稿；冻结原则 2026-07-03 拍板）
- **来源**：设计文档 4.1.2、附录《知识类型模板全集》；技术设计文档 1.2 变更②

## 背景

模板若上线后持续迭代，存量知识需要重整、校验器需要多版本并存，维护成本远超收益。

## 决策

六类类型（faq / sop / policy / product / case / term）与模板按设计文档现版**定稿冻结**（原"待业务方确认"解除）；不做模板版本迭代与存量重整。校验器与模板一起冻结为纯函数，改动走发版。

## 理由

模板是解析器、校验器、hash 规范化三者的共同契约，冻结后三者可放心硬编码；确需改模板时按新类型演进。
