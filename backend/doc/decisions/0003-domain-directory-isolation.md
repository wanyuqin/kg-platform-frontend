# ADR-0003：domain 白名单映射 viking:// 一级目录实现权限隔离

- **状态**：已采纳
- **日期**：2026-07-03（设计评审期拍板）
- **来源**：设计文档 5.1

## 背景

不同业务域的知识需要权限隔离，且检索时不能泄漏白名单之外的内容。

## 决策

每个 domain 对应 `viking://resources/{domain}/` 一级目录；API Key 的 domain 白名单在检索时映射为目录前缀集合（自动并入 common），传给 OpenViking 限定检索范围。

## 理由

权限边界 = 目录前缀，检索时天然物理隔离，不依赖检索后过滤，无泄漏面。
